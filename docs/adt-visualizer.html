코드로 배우는 자료구조 시각화

C++과 비슷한 간단한 코드를 입력하면 배열과 연결 리스트의 변화 과정을 단계별로 확인할 수 있습니다. 실제 C++ 컴파일러가 아니라, 학습용으로 제한된 문법만 해석합니다.

1

자료구조 선택

Array 또는 Linked List 탭을 선택하면 예시 코드와 설명이 자동으로 바뀝니다.

2

코드 수정

예시 코드를 그대로 실행하거나 숫자와 index를 바꿔서 직접 실험할 수 있습니다.

3

단계별 실행

Run 버튼을 누르면 각 연산이 1초 간격으로 재생되고, 현재 변화 위치가 강조됩니다.
